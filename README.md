### Add News Api
---
You need to create a [News API](https://newsapi.org/) key and add it to the strings.xml file

### Features
---
- 5 different language resources.
- 6 news categories.
- News search.
- News detail.
- Open the news source URL.

### Application Architecture
---
- IDE: Android Studio
- API: News API
- Arthitecture: MVVM
- Programming Language: Java
